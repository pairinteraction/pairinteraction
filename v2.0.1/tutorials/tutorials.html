Tutorials

The pairinteraction software comes with a graphical user interface (GUI) and a Python API. The GUI provides a quick way to calculate Rydberg interaction potentials, lifetimes, and energy shifts in the presence of external fields. The Python API is designed to automatize calculations and to have more fine-grained control.
